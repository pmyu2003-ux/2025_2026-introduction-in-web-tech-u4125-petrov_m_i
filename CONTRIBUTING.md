# Contributing to DevOps Lab Project

## Как сообщить об ошибке
- Убедитесь, что проблема ещё не была описана в Issues.
- Создайте новый Issue с подробным описанием шагов для воспроизведения.

## Предложение изменений
1. Сделайте форк репозитория.
2. Создайте ветку для своей функции (feature-branch).
3. Внесите изменения и сделайте коммит с понятным сообщением.
4. Откройте Pull Request в ветку `develop`.

## Стиль кода
- Следуйте общепринятым стандартам для используемого языка.
- Добавляйте комментарии, где это необходимо.
